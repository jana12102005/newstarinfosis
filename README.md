## ⚙️ Platform Overview

This web application provides:

* 🔐 User authentication (Login/Register with verification)
* 👨‍💼 Admin dashboard for managing users, projects, and requests
* 📦 Project listing and ordering system
* 💡 Idea submission system for custom development
* 📬 Email notification system
* 📊 Structured backend using Flask and SQLAlchemy

---

## 🛠️ Tech Stack

* **Backend:** Flask (Python)
* **Database:** MySQL (Aiven)
* **ORM:** Flask-SQLAlchemy
* **Frontend:** HTML, CSS, JavaScript
* **Deployment:** Render (Gunicorn)
* **Email Service:** SMTP (Gmail)

---

## 🚀 Deployment

This project is configured for deployment on **Render** using:

* `gunicorn` as the production server
* Environment variables for sensitive data
* External MySQL database (Aiven)
